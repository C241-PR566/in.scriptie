# API Documentation

## BASE URL
The base URL for all endpoints is https://capstone-project-c241-pr566.et.r.appspot.com/

## Endpoints

### User Registration

- **Endpoint:** `/register`
- **Method:** `POST`
- **Description:** Register a new user.

### User Login

- **Endpoint:** `/login`
- **Method:** `POST`
- **Description:** Authenticate user and return JWT token.
